# Дизайн-направление SiteBrain

## Общий стиль

Лендинг должен выглядеть как современный B2B IT-сайт: уверенно, чисто, технологично и без визуального шума. Главная задача дизайна — помочь бизнес-клиенту быстро понять услуги, довериться команде и перейти к обсуждению проекта.

## Визуальные принципы

- Тёмный фон: near-black / graphite.
- Молочный или белый основной текст.
- Один аккуратный акцентный цвет: blue, cyan или green.
- Строгие карточки с понятной иерархией.
- Крупные заголовки без чрезмерной декоративности.
- Аккуратные границы и разделители.
- Лёгкие gradients и blur использовать умеренно.
- Минимум анимаций, только для улучшения восприятия.
- Без кислотного hacker-style.
- Без перегруза glow-эффектами.
- Обязательная адаптация под телефон.

## Layout

- Использовать широкие секции с ограниченным контейнером.
- Не превращать каждую секцию в отдельную большую плавающую карточку.
- Карточки использовать для услуг, пакетов, портфолио и шагов процесса.
- На мобильных устройствах карточки должны идти в одну колонку.
- CTA должен быть заметным, но не кричащим.

## Типографика

- Заголовки короткие, крупные, с ясной пользой.
- Основной текст — спокойный, читаемый, без мелкого серого текста.
- Не использовать отрицательный letter-spacing.
- Не масштабировать шрифты напрямую от ширины viewport.

## Интерактив

- Hover states должны быть мягкими: border, background, лёгкий translate или accent glow.
- Не использовать сложные анимации в v1.
- Все интерактивные элементы должны быть доступны с клавиатуры.
- На мобильных устройствах кликабельные зоны должны быть достаточно крупными.
